# LOL ワードウルフ ⚔️

League of Legends（LOL）をテーマにしたワードウルフゲームのウェブアプリケーションです。友達と一緒にオンラインで楽しめるパーティーゲームです！

## 🎮 ゲーム概要

ワードウルフは、プレイヤーの中に1人だけ「少し違うお題」を持つ人（ウルフ）がいる推理ゲームです。話し合いを通じて、誰がウルフかを見破りましょう！

### 基本ルール

1. 各プレイヤーにお題が配られます
2. 制限時間内でお題について話し合います
3. 時間になったら、誰がウルフか投票します
4. 最多票を獲得したプレイヤーが処刑されます
5. **処刑されたのがウルフなら市民の勝利！**
6. **ウルフが生き残ったらウルフの勝利！**

### LOLテーマの例

- **市民のお題**: ヤスオ
- **ウルフのお題**: ヨネ

## ✨ 機能

### ゲーム機能
- 🏠 **ルーム作成・参加** - 6桁のルームIDまたはURLで簡単に友達を招待
- 👥 **3〜5人対応** - 柔軟なプレイ人数設定
- ⏱️ **タイマー機能** - 1〜15分のカスタマイズ可能な討論時間
- 💬 **リアルタイムチャット** - ゲーム中に議論できるチャット機能
- 🗳️ **投票システム** - 誰がウルフかを投票
- 🎯 **結果表示** - 勝敗判定と役割公開

### お題データ（全73ペア）
- **チャンピオン**: 20ペア（例: ヤスオ vs ヨネ、リー・シン vs ブラインドモンク）
- **アイテム**: 15ペア（例: インフィニティエッジ vs ストームレイザー）
- **スキル・能力**: 15ペア（例: ブリンク vs ダッシュ、スタン vs ルート）
- **マップ・レーン**: 15ペア（例: トップレーン vs ミッドレーン）
- **スペル**: 8ペア（例: フラッシュ vs ゴースト、イグナイト vs テレポート）

### デザイン
- 🎨 LOL公式風のダークテーマ
- 🌈 ティールグリーン × ゴールドの配色
- 📱 スマホ・PC両対応のレスポンシブデザイン
- ✨ 滑らかなアニメーション効果

## 🚀 使い方

### 1. ホストがルームを作成

1. 「ルームを作成」ボタンをクリック
2. プレイ人数、お題のカテゴリー、討論時間を設定
3. ルームIDまたは共有URLを友達に送信

### 2. 参加者がルームに参加

1. 「ルームに参加」にルームIDを入力
2. または、共有URLから直接アクセス

### 3. ゲーム開始

1. 全員が揃ったらホストが「ゲーム開始」をクリック
2. 各自に配られたお題を確認
3. チャットで話し合い

### 4. 投票

1. 討論時間が終わったら「投票を開始」をクリック
2. ウルフだと思うプレイヤーに投票

### 5. 結果確認

1. 勝敗と各プレイヤーの役割が公開されます
2. 投票結果も確認できます

## 🛠️ 技術スタック

### フロントエンド
- **HTML5** - セマンティックマークアップ
- **CSS3** - モダンスタイル、レスポンシブデザイン

- ✅ アクセシビリティ対応（ARIA属性、キーボード操作）

## 🔧 カスタマイズ

### お題を追加する

`js/data.js`の`TOPIC_PAIRS`オブジェクトに新しいペアを追加できます：

```javascript
const TOPIC_PAIRS = {
    champion: [
        { citizen: "新チャンピオン1", wolf: "新チャンピオン2" },
        // ...
    ]
};
```

### 設定を変更する

`js/game.js`の`GameState`クラスでデフォルト設定を変更できます：

```javascript
this.settings = {
    playerCount: 4,        // デフォルトプレイ人数
    categories: [...],     // デフォルトカテゴリー
    discussionTime: 180    // デフォルト討論時間（秒）
};
```

## 🐛 トラブルシューティング

### ルームに参加できない
- ルームIDが正しいか確認してください
- ルームが満員の可能性があります
- ブラウザのLocalStorageが有効か確認してください

### 画面が更新されない
- ページを再読み込みしてください
- ブラウザのキャッシュをクリアしてください
- LocalStorageをクリアしてください（注意: ルームデータが削除されます）

### LocalStorageをクリアする方法
```javascript
// ブラウザの開発者ツール（F12）のコンソールで実行
localStorage.clear();
location.reload();
```
