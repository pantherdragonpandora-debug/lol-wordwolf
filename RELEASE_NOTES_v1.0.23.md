## 🎮 新機能: ヴォイドに届くは光か闇か

連想ワードを伝えてテーマを当てる新しいゲームモードを追加しました！

### ゲーム概要

```
1. 最初のプレイヤーにテーマが表示される
   例: 「アッシュ」

2. そのプレイヤーが連想ワード3つを入力
   例: 「氷」「弓」「フレイヨルド」

3. 次のプレイヤーは前の3ワードを見て：
   - 伝わりにくいワードを0〜3個修正可能
   - 新しい3ワードを入力

4. 最後のプレイヤーが元のテーマを推測して回答

5. 正解なら成功！
```

---

## ✨ 特徴

### 1. 修正機能
- 前のプレイヤーの言葉を0〜3個修正できる
- どれを修正したかが結果で分かる
- 伝言ゲームの面白さを楽しめる

### 2. 豊富なテーマ
- **League of Legends**: チャンピオン、アイテム、場所、概念など25テーマ
- **VALORANT**: エージェント、武器、マップ、概念など15テーマ
- 今後も追加予定

### 3. 柔軟な人数設定
- **2〜8人**で遊べる
- 少人数でも大人数でも楽しめる

### 4. 美しいUI
- ヴォイドをイメージした紫/青のグラデーション
- アニメーション効果
- レスポンシブデザイン（スマホ対応）

## 🧪 テスト手順

### 基本フロー
```
1. モード選択 → ヴォイドを選択
2. ゲームタイプ選択 → LOL または VALORANT
3. ルーム作成
   - プレイヤー名入力
   - 人数選択（2〜8人）
   - テーマ選択（ランダム）
4. ルーム作成成功 → 待機画面表示
5. 他のプレイヤーが参加（別ブラウザ）
6. ホストがゲーム開始
7. 最初のプレイヤー: テーマを見て3ワード入力
8. 中間プレイヤー: 前の3ワードを見て修正＆新3ワード入力
9. 最後のプレイヤー: 3ワードから回答
10. 結果表示: 正解・不正解、推移を表示
```
