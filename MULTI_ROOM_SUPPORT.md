# 気分診断チャンピオン選択機能 (v1)

## 📅 実装日
2026-02-20

## 🎯 概要

**League of Legendsの気分診断チャンピオン選択ツール**を実装しました！

プレイヤーの今の気分や好みに応じて、最適なチャンピオンをおすすめする診断ツールです。

### コンセプト
- 「今日はどのチャンピオンを使おうかな？」という悩みを解決
- 5つの簡単な質問に答えるだけ
- アグレッシブ、サポーティブ、タクティカル、バランス型に分類
- ゲーム知識がなくても直感で答えられる質問

---

## 🎭 機能詳細

### 1. 診断フロー

```
1. ホーム画面
   ↓
2. 5つの質問
   - 今日の気分は？
   - どんなプレイスタイルが好き？
   - 好きなレーンは？
   - チームファイトでの役割は？
   - 今の気持ちに一番近いのは？
   ↓
3. スコア計算
   - aggressive (攻撃的)
   - supportive (支援的)
   - tactical (戦術的)
   ↓
4. 結果表示
   - タイプ診断
   - スコアチャート
   - おすすめチャンピオン3体
```

## 📊 スコアリングシステム

### 質問と配点
各質問の選択肢には以下のポイントが割り振られています：

```javascript
{
  aggressive: 0-3点,
  supportive: 0-3点,
  tactical: 0-3点
}
```

### 判定ロジック
```javascript
1. 5つの質問の合計スコアを計算
2. 最も高いスコアのタイプを判定
3. スコアが均等な場合はバランス型
4. おすすめチャンピオンをランダムに3体選択
```
