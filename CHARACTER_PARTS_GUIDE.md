# 🎨 セリフ当てクイズ - 画像パーツの使い方ガイド

## 📋 概要

このゲームでは、ユーザーが提供した **45枚の画像パーツ** を組み合わせて、3,125通りのユニークなキャラクターを生成します。

---

## 🗂️ 画像ファイル配置先

すべての画像は `images/` フォルダに配置してください。

### 必要な画像一覧

| カテゴリー | ファイル名 | 枚数 |
|----------|----------|------|
| 背景 | bg1.png ~ bg5.png | 5枚 |
| ボディ | body1.png ~ body25.png | 25枚 |
| 目 | eye1.png ~ eye5.png | 5枚 |
| 口 | mouth1.png ~ mouth5.png | 5枚 |
| アイテム | item1.png ~ item5.png | 5枚 |
| **合計** | - | **45枚** |

---

## 📐 画像仕様

- **推奨サイズ**: 400×400px
- **ファイル形式**: PNG（透過対応）
- **背景**: 
  - 背景画像（bg1~5）: 不透過OK
  - その他（ボディ、目、口、アイテム）: **透過PNG推奨**
- **カラー**: LoLの世界観に合わせたファンタジー風推奨

---

## 🎨 描画順序

キャラクター生成時、以下の順序で重ねて描画されます：

1. **背景** (bg1~5) - 最背面
2. **ボディ** (body1~25)
3. **目** (eye1~5)
4. **口** (mouth1~5)
5. **アイテム** (item1~5) - 最前面

※ Canvasの `drawImage()` を使用し、全て400×400pxに拡大/縮小されます。

---

## 🔄 フォールバック機能

画像が見つからない場合、自動的に **Canvas描画のプレースホルダー** が表示されます。

### プレースホルダー仕様
- 背景: グラデーション（5色パターン）
- ボディ: 図形（円、四角、三角、楕円、ひし形）× 5色 = 25パターン
- 目: 5種類の表情（丸目、つり目、大目、点目、X目）
- 口: 5種類の表情（笑顔、直線、悲しい、O形、ニヤリ）
- アイテム: 5種類（王冠、帽子、リボン、星、眼鏡）

---

## 🚀 実装の仕組み

### 1. 画像の事前読み込み

```javascript
// ページ読み込み時に全45枚を非同期で読み込み
await preloadImages();
```

### 2. キャラクター生成

```javascript
// ランダムに組み合わせを決定
const combinationId = `body05_eye03_mouth02_item01`;

// Canvasに描画
drawCharacter('character-canvas', combinationId);
```

### 3. 描画処理

```javascript
// 背景 → ボディ → 目 → 口 → アイテム の順に重ねる
drawImagePart(ctx, canvas, 'bg', bgNum);
drawImagePart(ctx, canvas, 'body', bodyNum);
drawImagePart(ctx, canvas, 'eye', eyeNum);
drawImagePart(ctx, canvas, 'mouth', mouthNum);
drawImagePart(ctx, canvas, 'item', itemNum);
```

## 📝 画像配置手順

### ステップ1: 画像ファイルを準備

ユーザーが提供した画像を以下の名前でリネームしてください：

```
bg1.png, bg2.png, bg3.png, bg4.png, bg5.png
body1.png, body2.png, ..., body25.png
eye1.png, eye2.png, eye3.png, eye4.png, eye5.png
mouth1.png, mouth2.png, mouth3.png, mouth4.png, mouth5.png
item1.png, item2.png, item3.png, item4.png, item5.png
```

### ステップ2: imagesフォルダにアップロード

プロジェクトの `images/` フォルダに全ファイルをアップロードします。

### ステップ3: 確認

`lol-quote-quiz.html` を開き、以下を確認してください：

1. コンソールに `✅ 全45枚の画像パーツを読み込みました` と表示される
2. キャラクター生成時に実際の画像が表示される

## 🎯 組み合わせ総数

**3,125通り** = 5 (背景) × 25 (ボディ) × 5 (目) × 5 (口) × 5 (アイテム)

同じ組み合わせが連続で生成されないようにローカルストレージで管理しています。

## 🐛 トラブルシューティング

### 問題: 画像が表示されない

**原因1**: ファイル名が間違っている
- 解決: 正確なファイル名（`body1.png`、`eye3.png` など）を確認

**原因2**: ファイルパスが間違っている
- 解決: `images/` フォルダ直下に配置されているか確認

**原因3**: ファイル形式が間違っている
- 解決: PNG形式であることを確認（JPG, GIF は使用不可）

**原因4**: ファイルが破損している
- 解決: 画像を別の画像エディタで再保存して試す

### 問題: 画像が小さく/大きく表示される

Canvas は自動的に400×400pxにリサイズします。
元画像のアスペクト比を保ちたい場合は、事前に400×400pxで作成してください。
