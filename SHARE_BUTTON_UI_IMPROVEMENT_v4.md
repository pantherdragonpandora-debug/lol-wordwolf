# シェアボタンのUI改善 (v4)

## 📅 更新日
2026-02-20

## 🎯 改善内容

### ユーザーフィードバックへの対応
「シェアボタンをもう少しわかりやすいアイコン（シェアと書くなど）にして。各SNSアイコンも色しかないので、わかりやすくして。」

### 実装した改善

#### 1. メインボタンの改善
**Before (v3)**
- 📤 絵文字のみ
- 丸いボタン（56px × 56px）
- テキストなし

**After (v4)**
- 🔗 シェアアイコン + 「シェア」テキスト
- 横長ボタン（100px × 48px）
- 一目で機能が分かる

#### 2. SNSボタンの改善
**Before (v3)**
- アイコンのみ（色で判別）
- 丸いボタン
- プラットフォーム名なし

**After (v4)**
- アイコン + プラットフォーム名
- 横長ボタン
- テキストラベル付き：
  - **X** - X (Twitter)
  - **Discord** - Discord
  - **LINE** - LINE
  - **コピー** - リンクコピー

## 🎨 新しいデザイン

### ビジュアル

```
閉じている状態：

│                    │
│              ┌────────┐

│              └────────┘

開いている状態：
┌────────────────────┐
│           ┌──────────┐

│           │ 📱 LINE   │

│           │ 🔗 シェア │
│           └──────────┘
└────────────────────┘
```


### CSS変更

#### ボタン形状
```css
/* Before: 丸いボタン */
.floating-share-toggle {
  width: 56px;
  height: 56px;
  border-radius: 50%;
}

/* After: 横長ボタン */
.floating-share-toggle {
  min-width: 100px;
  height: 48px;
  border-radius: 24px;
  padding: 0 16px;
}
```

#### テキスト表示
```css
.floating-share-btn span {
  white-space: nowrap;  /* テキストを折り返さない */
  font-size: 13px;
  font-weight: 600;
}
```

#### ホバーエフェクト
```css
/* Before: 拡大 */
.floating-share-btn:hover {
  transform: scale(1.1);
}

/* After: 左にスライド */
.floating-share-btn:hover {
  transform: translateX(-4px);
}
```
